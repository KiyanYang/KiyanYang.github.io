Markdown 语法 拓展篇

本文最后更新于：3 小时前

当 Markdown 无法满足一些高级需求时，可以直接插入 html 代码满足需求。这并不是 Markdown 的语法，只是可以在 Markdown 里使用。下面列举一些常用的功能。

颜色

  • 效果:

    #123ABC 颜色

  • Markdown

大小

  • 效果:

    0.8em 文字大小

  • Markdown

位置

  • 效果:

    居中

  • Markdown

    位置可选值：left | center | right。

折叠

  • 效果:

    点击查看效果
    左对齐居中对齐右对齐
    A1B1C1
    A2B2链接

  • Markdown

    summary 填写显示名称。

  • 注意

    在内部使用 Markdown 语句时要在前后留有空行，否则可能会当作普通文本处理，而且某些渲染器可能不支持在 html 内写 Markdown 文本，不支持时可以直接使用 html。

图片

  • 效果:

  • Markdown

    width 和 height 可全写、或只写 1 个、或全部不写，其值可为数字或百分比。在外部使用位置标签

    备注：图片居中显示是博客的效果，一般情况下是靠左显示，不过可以在其外面包裹位置，来调整图片的水平位置。

表格

由于博客的主题样式，表格的边框、背景等元素可能与你使用下列代码时的效果不同。

普通

  • 效果:

    111213
    212223
    313233

  • Markdown

纵向合并单元格

  • 效果:

    111213
    212223
    3233

  • Markdown

横向合并单元格

  • 效果:

    111213
    2123
    313233

  • Markdown
