HashTool 哈希校验工具#

应用平台：Windows

用于计算和校验文件、文件夹或文本哈希值的工具！

A Tool for Calculating and Verifying the Hash Value of Any File, Folder, or Text!

HashTool 支持文件、文件夹或文本的哈希计算，此外还提供了结果导出和对比框校验功能。

HashTool 支持多种算法：

  • CRC32
  • MD4、MD5
  • SHA1
  • SHA2 系列
  • SHA3 系列
  • Blake2B 系列
  • Blake2S 系列
  • Keccak 系列
  • QuickXor

项目地址：GitHub（主要），Gitee（镜像）。
